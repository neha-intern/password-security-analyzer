# 🔐 Password Security Analyzer

## 📌 Overview
This project is a simple Python-based security tool that analyzes password strength and checks whether a password has been exposed in common leaked databases.

It helps users understand how secure their passwords are and encourages better password practices.

---

## 🚀 Features
- Checks password strength (Weak / Moderate / Strong)
- Validates:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Detects common weak passwords
- Simulates password breach check
- Gives improvement suggestions

---

## 🛠️ Tech Used
- Python 3
- Regular Expressions (re module)
